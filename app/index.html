Welcome to ViewCast

Here’s the fastest way to get a new account ready for your first service.

1
Connect your data

Paste your Firebase web config once. If shared cloud sign-in is unavailable, Starter Mode keeps this device running so you can keep building.

2
Open Admin and load content

Add songs, media decks, background loops, typography, and Cloudinary uploads from the Admin area.

3
Build a playlist

Create a playlist, add song and media segments, then re-order them to match your service flow.

4
Go live with confidence

Open FOH and BOH, send cues with the keyboard, control video timelines, and auto-cycle sermon slides when needed.

Step-by-step help is always available from the left sidebar or the full guide. This tutorial only appears once per account unless you open it again manually.
